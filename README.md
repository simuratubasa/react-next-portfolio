# README

# react-next-portfolio

サイトURL: https://react-next-portfolio-umber-ten.vercel.app/

シンプルなポートフォリオサイトのサンプルです。

# 課題の紹介

## TOP

- ヒーローセクションで概要を表示
- 最新記事の導線を配置

## 自己紹介のページ

- 経歴の要約を掲載
- スキル一覧を簡潔に整理

## ブログ一覧ページ

- カテゴリ別に表示
- ページネーション対応
- 検索フォームを配置

## ブログ詳細ページ

- 記事本文の読みやすさ重視
- 公開日の表示
- 関連記事へのリンク

# 一番見てほしいところ

- 全体の情報設計
- 読みやすいレイアウト
- 直感的な導線

## 実装面

- App Routerを用いた構成
- コンポーネント分割
- microCMS連携を想定

## デザイン面

- 余白とタイポグラフィ
- シンプルで統一感のある配色
